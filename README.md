# Olympics_Data_Analysis_webapp
An interactive web application built using Streamlit to analyze historical Olympic Games data. The project provides insightful visualizations and statistics on medal tallies, country-wise performance, athlete demographics, and gender participation trends across different Olympic editions.
## Olympics Data Analysis Web App

An interactive Streamlit-based web application to analyze Olympic Games data
including medal tally, country-wise performance, athlete statistics, and
gender participation trends.

### Features
- Medal tally analysis by year and country
- Country-wise performance trends
- Athlete height vs weight analysis
- Gender participation over time
- Interactive visualizations using Plotly and Seaborn

### Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Plotly, Matplotlib, Seaborn

### Deployment
Deployed on Streamlit Cloud
